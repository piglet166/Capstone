Name:
  Gork Rework
Version:
  0.1-1
Description:
  Kemono's gork sprite reworked as a drop-in replacement for
  Stendhal's snarfkin creature.
Sprite details:
  - Orientation: orthogonal
    - N/E/S/W format for Stendhal.
    - S/W/E/N format for RPG Maker & RPGBoss.
  - Dimensions: 16x16, 16x24, 32x32, 32x48
  - Animation:
    - idle (center frames)
    - walking: 12 frames (3 per direction)
    - Animated preview @ 150ms delay.
  - PNG images use indexed color (smaller files).
  - GIMP sources (.xcf) use RGB color.
